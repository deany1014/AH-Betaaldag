# AH Betaaldag Aftelling 🏪💰

Een moderne, interactieve countdown website speciaal voor Albert Heijn medewerkers. Aftellen naar betaaldag is nog nooit zo mooi geweest!

## 🎨 Functies

- **Live Countdown Timer** - Realtime aftelling naar de volgende betaaldag (25e van de maand)
- **Smooth Animaties** - Pulserende getallen, blinkende separators en fade-in effecten
- **Modern Design** - Gradient achtergrond, glasmorphism effecten en responsive layout
- **Interactive Elements** - Subtiele 3D perspectief effecten bij muisbeweging
- **Responsive** - Perfect op desktop, tablet en mobiele apparaten
- **Nederlandse Interface** - Volledig in het Nederlands

## 📋 Wat is Betaaldag?

Voor AH medewerkers is betaaldag standaard op de **25e van elke maand**. Deze website telt af naar je volgende salaris!

## 🚀 Hoe te Gebruiken

1. Open `index.html` in je webbrowser
2. Klik op het bestand en open het met je favoriete browser
3. Geniet van de mooie countdown naar je betaaldag!

## 🎨 Design Features

- **Gradient Achtergrond** - Bewegende paarse en roze gradient
- **Glasmorphism Cards** - Semi-transparante kaarten met blur effect
- **Number Pulse Animation** - Animatie wanneer getallen veranderen
- **Blinking Separators** - Uurglaseffect met blinkende dubbele punten
- **3D Tilt Effect** - Subtle perspectief effect wanneer je over kaarten beweegt
- **Motivational Message** - Bemoedigende tekst voor hardwerkende medewerkers

## ⚙️ Aanpassen

### Betaaldag Datum Wijzigen

Open `script.js` en zoek naar `paydayDate = new Date(currentYear, currentMonth, 25);`

Verander `25` naar een ander getal (bijv. `15` voor mid-month betaling).

### Kleuren Wijzigen

Open `styles.css` en wijzig de gradient kleuren:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
